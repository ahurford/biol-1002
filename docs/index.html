Overview: In this week’s lab, we will be building on the R skills you learned in BIOL 1001. We will be focusing on manipulating and organizing data in tables, and learn a useful package called “dplyr” (pronounced “d-plier”)

You will need:

  • The following .csv files from Brightspace

    • shrub-volume-data.csv

    • gainesville-precip.csv

    • surveys.csv
